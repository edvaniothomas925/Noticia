1) Conteúdo original: AdSense costuma reprovar sites com cópias diretas. Se usar feeds, sempre gere resumos originais e adicione valor (comentários, análise).
2) Política de privacidade, página Sobre e Contato: crie estas páginas e links no rodapé. AdSense exige política de privacidade.
3) Velocidade: compacte imagens, use cache, e hospede o JSON numa CDN se possível.
4) Sitemap & robots.txt: gere sitemap.xml e robots.txt apontando para sitemap.
5) Mobile-friendly: template é responsivo; teste no Mobile-Friendly Test do Google.
6) Estrutura de heading e meta tags: cada artigo individual (se fizer páginas por artigo) deve ter título único, meta description e tags og:*
7) Evite conteúdo sensacionalista, spam e duplicado. Poste regularmente, mantenha qualidade.
8) Monetização: adicione blocos de anúncios depois que a conta AdSense estiver aprovada. Não coloque muitos anúncios acima da dobra.
